# backend/prompts/outputs/

Output dump for `backend/prompts/run-prompts.ts` — a small script that
runs each prompt against `sample-user.json` and writes the model's reply
here for inspection.

This directory is intentionally empty in git. Run `npx tsx run-prompts.ts`
from `backend/prompts/` to populate it. The contents are gitignored
artifacts, not source.
